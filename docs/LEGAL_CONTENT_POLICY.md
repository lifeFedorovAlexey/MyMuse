# Legal and Content Responsibility Policy (Draft)

## 1. Роль сервиса

MyMuse предоставляет программный инструмент для самостоятельного хранения, организации, стриминга и шаринга аудиофайлов, размещенных пользователями на их собственной инфраструктуре.

## 2. Ответственность пользователя

Пользователь самостоятельно несет ответственность за:

- законность хранения и распространения загружаемых треков;
- наличие прав, лицензий и разрешений;
- соблюдение локального законодательства при передаче ссылок и доступа третьим лицам.

## 3. Ограничение ответственности разработчика ПО

Разработчик/поставщик MyMuse:

- не предоставляет музыкальный каталог;
- не выступает издателем или дистрибьютором пользовательского контента;
- не несет ответственность за загруженные пользователем файлы.

## 4. Рекомендуемые меры для инстансов

- Добавить пользовательское соглашение при первом запуске.
- Хранить audit log действий загрузки/шаринга.
- Предусмотреть механизм удаления контента по запросу владельца инстанса.

## 5. External Import (будущий модуль)

Если будет реализован поиск/импорт из внешних источников, он должен:

- работать в рамках доступных API и лицензий;
- явно показывать источник контента;
- блокировать обход ограничений провайдеров.
